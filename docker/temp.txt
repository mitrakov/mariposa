Spark 4.1.1 bundles Hive
2.3.10 by default for metastore connectivity when enabled, supporting versions 2.0.0 through 4.0.1. For optimal performance, it is recommended to use the default 2.3.10 or configure a specific version using spark.sql.hive.metastore.version and spark.sql.hive.metastore.jars. 

    Default Version: Hive 2.3.10.
    Supported Versions: 2.0.0–2.3.10, 3.0.0–3.1.3, and 4.0.0–4.0.1.
    Configuration: Set spark.sql.hive.metastore.version in your spark-defaults.conf or SparkSession to match your external metastore. 

If using an external Hive Metastore (HMS), ensure the version matches the jars provided to Spark to avoid deserialization errors.
